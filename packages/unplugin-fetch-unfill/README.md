---

Install `unplugin-fetch-unfill` with your package manager.

Add the plugin to your config's plugin array:

```ts
import { fetchUnfillUnplugin } from "unplugin-fetch-unfill"

// ...
plugins: [
  // ...
  fetchUnfillUnplugin.{BUNDLER}()
]
```
